# 🚀 How to Use

Clone the repository

```bash
git clone https://github.com/Swapnil1105-arc/LeetCode_using-_pythonAndcpp.git
```

Open the folder

```bash
cd LeetCode_using-_pythonAndcpp
```

Navigate to any solution and run it.

Python

```bash
python solution.py
```

C++

```bash
g++ solution.cpp
./a.out
```
